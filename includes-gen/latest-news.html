Notenik 7.2.0 - Some Nifty Enhancements

Version 7.2.0 of Notenik includes a handful of nifty changes: the website address has been updated to point to Notenik.app; wiki links now will try to pluralize a reference, if it is not found in singular form; the Merge template Set command now allows a variable to be used as part or all of a global variable name; Markdown quote sharing corrected to respect lookup links; added a Quick Export and Open command.
