11.8.2 Phone

The word Phone can be used as both a label and a type.

A Phone field is meant to contain a single telephone number.

The user may enter a phone value with any desired spacing and punctuation. Only the plus sign and the comma, if entered, will have any significance.

When a phone field is displayed, it should result in a clickable link that will attempt to call the indicated phone number, when clicked.

When saved to disk, a phone value will be saved using the normal Markdown link syntax, using the tel URI scheme. The intent here is to make the resulting phone number clickable when the Note is opened in another application (such as iA Writer).
