6.2.11 Task Lists

As an extension to the original Markdown spec from John Gruber, task lists (also referred to as checklists and todo lists) are supported.

This is an extension documented in the Markdown Guide.

To create a task list, add dashes (-) and brackets with a space ([ ]) in front of task list items. To select a checkbox, add an x in between the brackets ([x]). 

- [x] Write the press release
- [ ] Update the website
- [ ] Contact the media

The rendered output will appear as follows on the Display tab.

  • ☑ Write the press release
  • ☐ Update the website
  • ☐ Contact the media
