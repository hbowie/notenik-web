14.2.2.6 The Work Type Slug

When a Work Type field is available for a Collection, then a special variable name of the-work-type-slug can be used in a Merge Template in order to obtain a nicely formatted phrase identifying the type of work.

If the Work Type field is missing, or blank, or has a value of ‘unknown’, then this derived variable will be blank. Otherwise, it will consist of the literal “ the ” followed by the work type value, converted to lowercase letters
