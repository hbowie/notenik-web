11.12.1 Address

The word Address can be used as both a label and a type.

Add an Address field to a Collection in order to store a street address that can be recognized by Apple Maps.

When the field is displayed, a custom URL for Apple Maps will be generated, and clicking on the link should open the address in the Apple Maps app.
