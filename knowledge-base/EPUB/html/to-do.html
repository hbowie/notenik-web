7.3.4 To Do

This is a Collection meant to be used as a simple Task list. It provides the following fields as defined in the Collection Template File. It is similar to the Web Links starter, with the addition of the Status, Date and Recurs fields.

Status: 0 - Idea; 1 - Proposed; 2 - Approved; 3 - Planned; 4 - In Work; 5 - Held; 6 - Completed; 7 - FollowUp; 8 - Canceled; 9 - Closed;
