22. You Want To Use VoiceOver

Notenik is written in Swift and uses AppKit and other standard macOS technologies.

This means that all of Notenik’s features are available using VoiceOver.

We didn’t realize how important – or how unusual – this was, until we started getting messages from self-described blind users expressing gratitude for their ability to use Notenik.
