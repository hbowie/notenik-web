2. Prepare Your Toolbox

You will need the following tools.

  1. Hardware

    • A reasonably modern Mac

  2. Software

    • Notenik
    • A text editor, such as BBEdit or Nova
    • An FTP app, such as Transmit

  3. Skills

    You need not be an expert at any of this, but you should at least have some basic understanding of:

    • Web servers - used to serve your website to the rest of the world
    • HTML - used to structure your web pages
    • CSS - used to style your website
    • Markdown - used to write your content.

  4. Web Stuff

    • Domain name - to provide an address for your website
    • Web hosting - this can be a shared server; I use A2 Hosting

  5. Reference

    • In addition to the links provided above, this guide will make frequent reference to the Notenik Knowledge Base. I’ll be providing links to the version published on the Web, but you can also feel free to access the version contained within Notenik, and accessible from the Help menu.
