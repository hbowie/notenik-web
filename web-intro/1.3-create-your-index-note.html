1.3. Create your Index note

Make sure you have the content / pages window frontmost within Notenik.

You should see an existing note with a title of Index, with the following body value. 

My name is Herb. I enjoy making websites, and 
making software to make it easier to make websites.

Go ahead and edit this note, changing the body to reflect your name and a little bit about yourself.

You will see that the pages collection offers multiple fields for each note.

Change the Header line to enter the text you wish to see at the top of your website’s homepage.

You can leave the Sub-Header field blank.

Make sure the Seq field is set to a value of 1.

Now save this note, by clicking on the Display tab, or by selecting Note > Save Note (or pressing Command-S).
