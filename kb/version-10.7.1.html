13.68 Version 10.7.1

Date: 08 Nov 2022 - Tuesday

Improved Tags Explosion Logic

Improved the handling of tags explosion when using this option as part of the Input Module Parameters available when playing a Script File.

Fixed a Bug in Link Interpretation

I recently implemented a means of creating Wiki Links Between Collections which involved using a slash to separate a shortcut or folder name from the title of the Note. Unfortunately, my logic was also looking for a slash when creating an ordinary link. This has been corrected.
