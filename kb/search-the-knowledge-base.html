2.3 Search the Knowledge Base

If you’re viewing the Knowledge Base as a Collection, within Notenik, then you can use Notenik’s Search function to search for a term of interest.

If you’re viewing the Knowledge Base from the web, then you can use this page, generated by the Markdown extension to Generate a Collection Search Page.
