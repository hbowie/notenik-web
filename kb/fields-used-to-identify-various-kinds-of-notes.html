5.13 Fields Used to Identify Various Kinds of Notes

The following field labels and types can be useful when you want to distinguish between several distinctly different kinds of Notes that may be present in a single Collection.

Each one of these field types has some unique functionality associated with it, so choose the one (or more) that will be most useful to you in a particular situation.
