9.2.3.15 Seq Formatted

This derived variable name can be used to obtain the formatted Seq value of a Note using formatting codes specified for the Seq field in the Collection Template file.

This derived variable can be specified as follows: 

=$seq-formatted$=

Also see the Seq Formatting - K variable modifier.
