13.6 Version 17.2.0

Date: 03 Jul 2025 - Thursday

Added Collection Sync

Notenik has a new Collection Sync command that can be used to synchronize data between two collections. This process also be script-driven, using the new Sync Module.

Added First Use Screen

Changed the First Use experience to allow a new user to quickly and easily create a simple collection of notes.
