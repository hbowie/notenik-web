5.3.8 Commonplace Book

This is a Collection meant to be used as a Commonplace book. It provides the following fields as defined in the Collection Template File. It makes use of Fields Useful for Reviews, Citations and Quotations. 

Title: <title> 

Tags: <tags> 

Link: <link> 

Timestamp:  

Author: <author> 

Work Type: <worktype> 

Work Title: <worktitle> 

Date: <date> 

Quote: <body>
