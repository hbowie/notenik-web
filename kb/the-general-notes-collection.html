6.2.2 The General Notes Collection

Use File > Make Collection General to designate the current collection as your General Notes collection.

This is intended to be a general-purpose collection for all sorts of notes that you’d like to quickly save.

Use File > Open General Notes or the associated Keyboard Shortcut to open your General Notes and/or bring it to the front.

Also see the Custom URL Scheme for ways to reference the General Notes collection via a Notenik URL. This can be used, along with a utility such as Keyboard Maestro, to create a keyboard shortcut that can be used from anywhere (not just from within Notenik).
