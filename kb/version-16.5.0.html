13.9 Version 16.5.0

Date: 03 Mar 2025 - Monday

Added Support for Custom Starter Packs

Notenik now makes it easy to augment its Built-In Starter Packs with custom packs created by a user, and the Knowledge Base now provides instructions on How To Create A Starter Pack. Additionally the term Starter Pack is now being used consistently to identify what used to be called a Starting Collection Type, or a Collection model.
