16. Version History ↑

16.20 Version 14.7.0

Date: 22 Jun 2024 - Saturday

Notenik Import Now Uses Import Settings

The useful Import Settings can now be applied when importing from another Notenik Collection. And these settings now include an option to consolidate lookup data.

Concatenated Markdown Can Now Include a Collection ToC

When exporting your Notes using the concatenated Markdown option, the Generate a Collection Table of Contents command will be expanded into a series of lines using ordinary Markdown syntax for lists and links.

Default Value Now Available When Adding a Field

When using the option to Rename, Add or Remove a Field, and when adding a new field, a default value for the new field may now be specified.
