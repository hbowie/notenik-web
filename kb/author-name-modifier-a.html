12.3.9 Author Name Modifier - A

The letter “A”, followed by a single digit, will cause the value to be treated as a person’s name, and to be reformatted as follows:

  • “1” - Extract the last name only
  • “2” - Place the last name first, followed by a comma, and then the rest of the name.
  • “3” - First name first, followed by last name, with no commas,
