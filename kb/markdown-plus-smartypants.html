4.1 Markdown plus SmartyPants

The Markdown parser that is native to Notenik generally follows the original Markdown Syntax Specification written by John Gruber, plus the SmartyPants typographic extensions.

Exceptions/extensions/additions to the original syntax are described in following pages, and grouped into categories.
