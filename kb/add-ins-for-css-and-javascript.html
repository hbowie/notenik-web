7.6 Add-Ins for CSS and Javascript

Starting with Version 16.1.0, a folder named add-ins may be placed within a collection folder. The add-ins folder can contain additional CSS and/or javascript files to be loaded. The CSS files must end with file extension .css, and the javascript files must end with the file extension .js.
